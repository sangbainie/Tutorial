# Windows安装（或重装）教程

此教程是基于Windows10和Windows11的，其他版本的Windows安装方法可能不同。  
Windows的常用安装办法有两种：

1. U盘安装介质安装
2. WinPE安装

## U盘介质安装

微软官方的安装方法非常简单，只需要准备一个容量大于8GB的U盘即可。

在[**Bing**](https://www.bing.com)搜索引擎中输入“下载Windows11”，进入微软官方的下载页面，找到“创建Windows11安装媒体”
![win11-U](/image/windows_download.png)
点击立即下载，下载好安装程序后，打开。
![win11-u-upset](/image/win11-U-upset.png)
一路来到这个页面
![win11upset](/image/win11uupset2.png)
选择U盘，下一步，选择你的U盘，下一步，等待Windows下载好后，就会制作安装介质，制作安装介质会**格式化这个U盘**

制作介质完成后，把U盘插到要安装系统的电脑上面，从U盘启动系统（每个品牌的主板方法不一样，自行搜索自己品牌的电脑如何从U盘启动系统），就来到了Windows官方的安装流程，跟着流程走即可
